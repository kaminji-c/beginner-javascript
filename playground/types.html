Objects

Are used for a collections of data, of functionality

Order doesn't matter in an Object. If you need order you use an Array

Symbol

A way to do unique properties or unique identifiers in JS

Undefined

Something that has been created, a variable but is yet defined

A variable that has not yet have a value set to it

Null

A value of nothing

Boolean

Is true or false

Use it for logic for if statements

Booleans can also be calculated

Can be a straight up value or a value stored in a variable

Equality

One equal sign is used to updating or setting a variable. Almost always use triple === Double equals are almost always bad practice

Triple ==== will always check both the value and the type
