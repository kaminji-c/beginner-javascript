Map is very similar to an Object. It is simply used to store data

Benefit of the map: 1. Map KEYS and values can be any type. In an Object only the value of the KEY can be any type.

Sometimes in JS we have these things called DICTIONARIES which is a way to store additional meta data

DOWNSIDES of a MAP: 1. No literal notation 2. You cannot store a function in it like an object can(methods) 3.JSON does not handle maps
