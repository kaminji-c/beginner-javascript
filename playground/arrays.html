Arrays

An array is used for holding a list of items where the ORDER matters. Each thing inside of an array is called an ITEM and its POSITION in an array is called an INDEX

NUMBERS inside of an array is called a LENGTH

EACH item inside of an array can be ANY TYPE

Similar to an object except there are NO KEYS

The KEYS will be INDEXES so 0, 1, 2, 3, 4, 5

Array is not its own TYPE, at the end of the day its an OBJECT

Best way to check if something is an array is using the Array.isArray (static) method: Array.isArray();

Arrays are zero based. Index counting starts at 0

Mutable Methods

Perform mutations. When the original version of something is changed it is considered a mutation

Immutable

The methods do not change the original thing but they simply return a new array
