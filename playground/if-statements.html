Truthy Values

  • * 1 - equates to true

  • * -10 -all numbers except 0 will be truthy

  • * full string

  • * a string of "0" - any value of a string will be truthy except an empty string

  • * empty array

  • * empty object

Falsy Values

  • * 0 - equates to false

  • * undefined variable

  • * Variable set to null

  • * a variable set to `"hello" - 10` NaN (not a number)

  • * empty string
